# 🛒 E-commerce Frontend com Bootstrap

Uma landing page responsiva para um e-commerce fictício desenvolvida com **HTML5** e **Bootstrap 5**. O projeto simula a página inicial de uma loja virtual moderna, apresentando banners promocionais, categorias de produtos, vitrines em destaque e um layout adaptado para diferentes dispositivos.

## 🌐 Demonstração

**Acesse o projeto online:**

👉 https://breninf.github.io/ecommerce-frontend-bootstrap/

---

## 📖 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos de desenvolvimento Front-End utilizando HTML semântico e componentes do Bootstrap.

A interface foi inspirada em grandes lojas virtuais, explorando recursos de responsividade, organização visual e experiência do usuário.

---

## ✨ Funcionalidades

- Barra de navegação responsiva
- Menu com categorias de produtos
- Barra de pesquisa
- Banner promocional
- Carrossel principal
- Vitrine de produtos femininos
- Vitrine de produtos masculinos
- Cards de produtos
- Rodapé institucional
- Ícones utilizando Bootstrap Icons
- Layout responsivo para diferentes tamanhos de tela

## 🚀 Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/Breninf/ecommerce-frontend-bootstrap.git
```

Entre na pasta do projeto:

```bash
cd ecommerce-frontend-bootstrap
```

Abra o arquivo `index.html` em qualquer navegador.

Ou utilize a extensão **Live Server** no Visual Studio Code para uma experiência de desenvolvimento mais prática.
